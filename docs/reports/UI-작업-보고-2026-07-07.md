- **2A:** `SearchBar`, `PaginationControls`, `DataTableCard`, `SortableTableHead`, `use-sort`, `use-pagination`

- Landing hero, `product-preview`, pricing

### Phase 6 — Polish + Performance

- **Workspace autosave** — `useWorkspaceDraft` + `AutosaveIndicator`
